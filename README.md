# c-cli-cc-ninja-system-call-hello-world

## Description
This is a POC project to demonstrate 
ninja as a build tool calling system
specific commands.

## Tech stack
- c
- cc
- ninja

## Docker stack
- alpine:edge

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
